## Developer Tools

Build powerful custom integrations:

### REST API

Full programmatic access to MailGreet:
- Create and manage subscribers
- Send campaigns via API
- Retrieve analytics data
- Manage templates and lists

**[View API Documentation →](/api-reference/introduction)**

### Webhooks

Receive real-time notifications:
- New subscriber events
- Email opens and clicks
- Unsubscribes and bounces
- Campaign completions

**[Learn about Webhooks →](/api-reference/endpoint/webhook)**

### Zapier

Connect MailGreet to 5,000+ apps without code:

<AccordionGroup>
  <Accordion title="Popular Zaps">
    - Google Sheets → MailGreet (import subscribers)
    - MailGreet → Slack (notify on new subscribers)
    - Stripe → MailGreet (add customers as subscribers)
    - Calendly → MailGreet (add booking contacts)
  </Accordion>
</AccordionGroup>
